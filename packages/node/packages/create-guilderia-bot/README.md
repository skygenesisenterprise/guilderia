## About

It's easy to create a simple Guilderia bot to begin your journey with the Guilderia API.

```sh
npm create guilderia-bot ./your/chosen/directory
yarn create guilderia-bot ./your/chosen/directory
pnpm create guilderia-bot ./your/chosen/directory
bun create guilderia-bot ./your/chosen/directory
```
