---
title: Formatters
---

guilderia.js provides the `formatters` package which contains a variety of utilities you can use when writing your Guilderia bot.

## Basic Markdown

These functions format strings into all the different Markdown styles supported by Guilderia.

```js
const { blockQuote, bold, italic, quote, spoiler, strikethrough, underline, subtext } = require('guilderia.js');
const string = 'Hello!';

const boldString = bold(string);
const italicString = italic(string);
const strikethroughString = strikethrough(string);
const underlineString = underline(string);
const spoilerString = spoiler(string);
const quoteString = quote(string);
const blockquoteString = blockQuote(string);
const subtextString = subtext(string);
```

## Links

There are also two functions to format hyperlinks. `hyperlink()` will format the URL into a masked markdown link, and `hideLinkEmbed()` will wrap the URL in `<>`, preventing it from embedding.

```js
const { hyperlink, hideLinkEmbed } = require('guilderia.js');
const url = 'https://guilderia.js.org/';

const link = hyperlink('guilderia.js', url);
const hiddenEmbed = hideLinkEmbed(url);
```

## Code blocks

You can use `inlineCode()` and `codeBlock()` to turn a string into an inline code block or a regular code block with or without syntax highlighting.

```js
const { inlineCode, codeBlock } = require('guilderia.js');
const jsString = 'const value = true;';

const inline = inlineCode(jsString);
const codeblock = codeBlock(jsString);
const highlighted = codeBlock('js', jsString);
```

## Timestamps

With `time()`, you can format Unix timestamps and dates into a Guilderia time string.

```js
const { time, TimestampStyles } = require('guilderia.js');
const date = new Date();

const timeString = time(date);
const relative = time(date, TimestampStyles.RelativeTime);
```

## Mentions

`userMention()`, `channelMention()`, and `roleMention()` all exist to format Snowflakes into mentions.

```js
const { channelMention, roleMention, userMention } = require('guilderia.js');
const id = '123456789012345678';

const channel = channelMention(id);
const role = roleMention(id);
const user = userMention(id);
```
